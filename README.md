# 📊 Análisis del Desempeño Financiero con SQL

## 📌 Descripción

En este proyecto analicé el desempeño financiero de Adventure Works utilizando SQL para calcular indicadores clave (KPIs), evaluar el rendimiento por país y generar recomendaciones basadas en datos.

## 🎯 Objetivo

Analizar la rentabilidad de diferentes países utilizando métricas financieras como ingresos, costos, beneficio bruto, margen y ROI para apoyar la toma de decisiones.

## 🛠 Herramientas utilizadas

- SQL
- Google Sheets
- Adventure Works

## 📈 KPIs analizados

- Ingresos
- Costos
- Beneficio Bruto
- Margen (%)
- ROI

## 📈 Resumen ejecutivo

El análisis permitió identificar qué países generan mayor rentabilidad y cuáles presentan mayores costos de campaña.

Los resultados muestran que Estados Unidos obtiene el ROI más alto mientras que algunos países europeos presentan costos elevados que reducen su eficiencia comercial.

## 💻 Consultas SQL destacadas

Durante el proyecto utilicé consultas SQL para:

- Calcular ingresos y costos por país.
- Obtener el Beneficio Bruto.
- Calcular Margen (%).
- Calcular ROI.
- Comparar el desempeño financiero entre países.

## 💡 Principales hallazgos

- Estados Unidos presentó el mayor ROI.
- Se identificaron diferencias importantes entre países.
- El aumento del gasto en campañas no garantiza un incremento proporcional del ROI.
